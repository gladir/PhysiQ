# Exemples Octopus

Scripts d'exemple pour l'interpreteur `OCTOPUS.EXE` (clone du langage
d'entree Octopus, octopus-code.org).

## Execution

Depuis le dossier racine du projet :

```
OCTOPUS.EXE SAMPLES\OCTOPUS\01_basics.inp
OCTOPUS.EXE -q SAMPLES\OCTOPUS\05_complex.inp
```

## Index des exemples

| Fichier | Description |
|---------|-------------|
| `01_basics.inp`   | Affectations, arithmetique, operateurs |
| `02_math.inp`     | Fonctions mathematiques (trig, log, exp, erf) |
| `03_units.inp`    | Constantes physiques et unites atomiques |
| `04_logic.inp`    | Operateurs logiques et de comparaison |
| `05_complex.inp`  | Nombres complexes |
| `06_blocks.inp`   | Blocs `%nom ... %` (geometrie, maille) |
| `07_include.inp`  | Directive `include` + `defs.inp` |
| `08_env.inp`      | Variables d'environnement |
| `09_h2_molecule.inp` | Exemple realiste : molecule H2 |
| `10_silicon.inp`  | Exemple realiste : cristal de silicium |
